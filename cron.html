Cron cheatsheet

Cron лучше всего подходит для планирования повторяющихся задач. Для однократных задач используйте утилиту at.

#Формат Crontab

#Формат

Мин  Час  День  Мес  ДеньНедели
*    *    *    *    *  команда для выполнения
┬    ┬    ┬    ┬    ┬
│    │    │    │    └─  День недели   (0=Вс .. 6=Сб)
│    │    │    └──────  Месяц         (1..12)
│    │    └───────────  День месяца   (1..31)
│    └────────────────  Час           (0..23)
└─────────────────────  Минута        (0..59)
Поле Диапазон Спецсимволы
Минута 0 - 59 , - * /
Час 0 - 23 , - * /
День месяца 1 - 31 , - * ? / L W
Месяц 1 - 12 , - * /
День недели 0 - 6 , - * ? / L #

#Примеры

Пример Описание
*/15 * * * * Каждые 15 минут
0 * * * * Каждый час
0 */2 * * * Каждые 2 часа
15 2 * * * В 2:15 (каждый день)
15 2 * * ? В 2:15 (каждый день)
10 9 * * 5 В 9:10 (каждую пятницу)
0 0 * * 0 В 00:00 только по воскресеньям
15 2 * * 1L В 2:15 в последний понедельник каждого месяца
15 0 * * 4#2 В 00:15 во второй четверг каждого месяца
0 0 1 * * В 00:00 1-го числа каждого месяца
0 0 1 1 * В 00:00 1 января
@reboot При каждой перезагрузке (нестандартно)

#Специальные строки

Спецстрока Значение
@reboot Один раз при загрузке системы (нестандартно)
@yearly Раз в год, "0 0 1 1 *" (нестандартно)
@annually (то же, что @yearly) (нестандартно)
@monthly Раз в месяц, "0 0 1 * *" (нестандартно)
@weekly Раз в неделю, "0 0 * * 0" (нестандартно)
@daily Раз в день, "0 0 * * *" (нестандартно)
@midnight (то же, что @daily) (нестандартно)
@hourly Раз в час, "0 * * * *" (нестандартно)

#Команда crontab

- -
crontab -e Редактировать или создать файл crontab, если его ещё нет.
crontab -l Показать файл crontab.
crontab -r Удалить файл crontab.
crontab -v Показать время последнего редактирования crontab (нестандартно).

#Специальные символы

Спецсимвол Описание
Asterik(*) Совпадает со всеми значениями в поле.
Hyphen(-) Диапазон. Напр.: 1-5 в 5-м поле (день недели) — с понедельника по пятницу
Slash (/) 1-е поле (минуты): /15 — каждые 15 минут или шаг диапазона.
Comma (,) Разделитель значений. Напр.: 2,6,8 во 2-м поле (час) — в 2:00, 6:00 и 8:00
L Только для дня месяца или дня недели. 2L в дне недели — последний вторник месяца
Hash (#) Только для дня недели, число 1–5. Напр.: 4#1 — первый четверг месяца.
Question mark (?) Вместо '*', только для дня месяца или дня недели. В выражении можно использовать только в одном из этих полей.
