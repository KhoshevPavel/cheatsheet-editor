Chmod cheatsheet

Краткая шпаргалка по правам доступа к файлам и команде chmod.

#Начало работы

#Синтаксис

$ chmod [options] <permissions> <file>

#Пример

$ chmod 755 foo.txt
$ chmod +x cheatsheets.py
$ chmod u-x cheatsheets.py
$ chmod u=rwx,g=rx,o= cheatsheets.sh

#Change files and directories recursively

$ chmod -R 755 my_directory

Команда chmod означает «change mode» (изменить режим доступа)

#Примеры chmod

#Операторы

Символ Описание
+ Добавить
- Remove
= Set

#chmod 600

$ chmod 600 example.txt
$ chmod u=rw,g=,o= example.txt
$ chmod a+rwx,u-x,g-rwx,o-rwx example.txt

#chmod 664

$ chmod 664 example.txt
$ chmod u=rw,g=rw,o=r example.txt
$ chmod a+rwx,u-x,g-x,o-wx example.txt

#chmod 777

$ chmod 777 example.txt
$ chmod u=rwx,g=rwx,o=rwx example.txt
$ chmod a=rwx example.txt

#Символический режим

Запретить выполнение для всех.

$ chmod a-x chmodExampleFile.txt

Разрешить чтение для всех.

$ chmod a+r chmodExampleFile.txt

Сделать файл читаемым и записываемым для группы и остальных.

$ chmod go+rw chmodExampleFile.txt

Сделать shell-скрипт исполняемым для владельца.

$ chmod u+x chmodExampleScript.sh

Разрешить всем читать, писать и выполнять файл и включить set group-ID.

$ chmod =rwx,g+s chmodExampleScript.sh

#Удаление разрешений

Чтобы снять права на чтение и запись у файла, используйте следующий синтаксис:

$ chmod o-rw example.txt

Для файла example.txt можно снять права на чтение и запись для группы командой:

$ chmod  g-rx example.txt

Чтобы снять права на чтение и запись у группы и добавить их для остальных, используйте:

$ chmod g-rx, o+rx example.txt

Чтобы снять все права для группы и остальных, используйте go=:

$ chmod go= example.txt

#Исполняемый файл

$ chmod +x ~/example.py
$ chmod u+x ~/example.py
$ chmod a+x ~/example.py

#chmod 754

$ chmod 754 foo.sh
$ chmod u=rwx,g=rx,o=r foo.sh

#Практики chmod

#SSH-разрешения

$ chmod 700 ~/.ssh
$ chmod 600 ~/.ssh/authorized_keys
$ chmod 600 ~/.ssh/id_rsa
$ chmod 600 ~/.ssh/id_rsa.pub
$ chmod 400 /path/to/access_key.pem

#Веб-разрешения

$ chmod -R 644 /var/www/html/
$ chmod 644 .htaccess
$ chmod 644 robots.txt
$ chmod 755 /var/www/uploads/
$ find /var/www/html -type d -exec chmod 755 {} \;

#Пакетное изменение

$ chmod -R 644 /your_path
$ find /path -type d -exec chmod 755 {} \;
$ find /path -type f -exec chmod 644 {} \;
