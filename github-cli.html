#Установка

Windows

Инструмент Установка Обновление
WinGet winget install --id Github.cli winget upgrade --id GitHub.cli
Scoop scoop install gh scoop update gh
Choco choco install gh choco upgrade gh

Mac OS

Инструмент Установка Обновление
Brew brew install gh brew upgrade gh
MacPorts sudo port install gh sudo port selfupdate && sudo port upgrade gh

Linux

См. инструкции по установке для других дистрибутивов Linux.

Скрипт установки:

type -p curl >/dev/null || (sudo apt update && sudo apt install curl -y)
curl -fsSL https://cli.github.com/packages/githubcli-archive-keyring.gpg | sudo dd of=/usr/share/keyrings/githubcli-archive-keyring.gpg \
&& sudo chmod go+r /usr/share/keyrings/githubcli-archive-keyring.gpg \
&& echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/githubcli-archive-keyring.gpg] https://cli.github.com/packages stable main" | sudo tee /etc/apt/sources.list.d/github-cli.list > /dev/null \
&& sudo apt update \
&& sudo apt install gh -y

#Основное использование

Справка и документация

Показать опции команды:

Команда Описание
gh help [command] Справка по любой команде. Введите gh help [путь к команде] для подробностей.

Аутентификация

Команда Описание
gh auth login Стандартная аутентификация через браузер
gh auth logout Удалить конфигурацию аутентификации для хоста
gh auth refresh Расширить или исправить scope прав для сохранённых учётных данных
gh auth setup-git Настроить git на использование GitHub CLI как credential helper
gh auth status Проверить и показать состояние аутентификации
gh auth token Вывести токен аутентификации, используемый gh

С токеном Github:

$ gh auth --with-token < token.txt

#Задачи (issues)

Поиск issues

Команда Описание
gh search issues [query] Поиск issues на GitHub

Примеры

# поиск issues по ключевым словам "readme" и "typo"
$ gh search issues readme typo

# поиск issues по фразе "broken feature"
$ gh search issues "broken feature" # поиск issues и PR в организации cli
$ gh search issues --include-prs --owner=cli

# поиск открытых issues, назначенных вам
$ gh search issues --assignee=@me --state=open

# поиск issues с большим числом комментариев
$ gh search issues --comments=">100" # поиск issues без метки "bug"
$ gh search issues -- -label:bug

#Pull requests

Действия с pull request

Команда Описание
gh pr create Создать pull request
gh pr list Список pull requests в репозитории
gh pr status Показать статус pull requests

$ gh pr status

  You have no open pull requests

  #15 New feature [branch]

#GitHub Actions

General Actions

Команда Описание
gh workflow disable Отключить workflow — не запускается и не показывается в списке
gh workflow enable Включить workflow — можно запускать и он показывается в списке
gh workflow list Список workflow-файлов (отключённые по умолчанию скрыты)
gh workflow run Создать событие workflow_dispatch для указанного workflow
gh workflow view Просмотр сводки workflow

Запуск Actions

Команда Описание
gh run cancel Отменить выполнение workflow
gh run delete Удалить выполнение workflow
gh run download Скачать артефакты выполнения GitHub Actions
gh run list Список последних выполнений
gh run rerun Перезапустить всё выполнение, только проваленные jobs или конкретный job
gh run view Просмотр сводки выполнения workflow
gh run watch Отслеживать выполнение до завершения с отображением прогресса
