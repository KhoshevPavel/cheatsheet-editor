Cheatsheet Editor

Локальный редактор HTML-шпаргалок. Откройте папку, выберите нужный файл плиткой и редактируйте его в TinyMCE с layout-кнопками.

Список файлов появится после выбора каталога.

Файл не выбран

Откройте папку `collector/cheatsheets/cheatsheets` и выберите шпаргалку слева.

Как пользоваться

1. Нажмите Открыть папку и выберите каталог со шпаргалками.

2. Слева появятся плитки всех `*.html` файлов.

3. Выберите файл, редактируйте визуально или в HTML и нажмите Сохранить.

Для записи обратно в файлы нужен Chromium-браузер с поддержкой File System Access API.

Ctrl/Cmd+F, Ctrl/Cmd+H, Alt+F
