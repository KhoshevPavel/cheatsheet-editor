Создаёт директорию проекта для нового консольного приложения

$ dotnet new console

Выводит список всех шаблонов приложений

$ dotnet new list

Компиляция и запуск (убедитесь, что находитесь в директории проекта)

$ dotnet run

#Строки C#

#Конкатенация строк

string first = "John";

#Строковая интерполяция

string first = "John";

#Строковые члены

Length Свойство, возвращающее длину строки.
Compare() Статический метод сравнения двух строк.
Contains() Проверяет, содержит ли строка указанную подстроку.
Equals() Проверяет, совпадают ли две строки посимвольно.
Format() Форматирует строку через нотацию {0} и примитивы.
Trim() Удаляет указанные символы в начале и конце. По умолчанию — пробелы.
Split() Разбивает строку по указанному символу и создаёт массив из частей.

#Основные термины .NET

Термин Определение
Runtime Набор служб для выполнения скомпилированной единицы кода.
Common Language Runtime (CLR) Загружает и управляет объектами .NET. Обрабатывает память, хостинг приложений, потоки, проверки безопасности и др.
Managed code Код, компилируемый и выполняемый в .NET. Примеры: C#, F#, VB.
Unmanaged code Код, компилируемый в машинный код, не управляется .NET. Без сборки мусора и т.п. Примеры: DLL из C/C++.
