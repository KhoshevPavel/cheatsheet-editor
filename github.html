Github cheatsheet

Наглядная шпаргалка по 80 горячим клавишам GitHub.com

#Горячие клавиши

#Ярлыки для всего сайта

Сочетание клавиш Действие
S Фокус на строку поиска
G N Перейти к уведомлениям
H Открыть карточку пользователя, issue или PR
Esc Закрыть карточку при фокусе на ней

#Репозитории

Сочетание клавиш Действие
G C Вкладка «Код»
G I Вкладка «Issues»
G P Вкладка «Pull requests»
G B Вкладка «Проекты»
G W Вкладка «Wiki»

#Редактирование исходного кода

Сочетание клавиш Действие
Ctrl F Поиск в редакторе файла
Ctrl G Найти далее
Ctrl Shift G Найти предыдущее
Ctrl Shift F Заменить
Ctrl Shift R Заменить всё
Alt G Перейти к строке
Ctrl Z Отменить
Ctrl Y Повторить

#Просмотр исходного кода

Сочетание клавиш Действие
T Поиск файлов
L Перейти к строке в коде
W Переключить ветку или тег
Y Развернуть URL в канонический вид
I Показать/скрыть комментарии в diff
B Просмотр blame

#Комментарии

Сочетание клавиш Действие
Ctrl B Жирный текст (Markdown)
Ctrl I Курсив (Markdown)
Ctrl K Ссылка (Markdown)
Ctrl Shift P Переключить вкладку Write/Preview
Ctrl Enter Отправить комментарий
Ctrl . Открыть сохранённые ответы и вставить в комментарий
Ctrl G Вставить предложение
R Цитировать выделенный текст

#Списки проблем и запросов на включение

Сочетание клавиш Действие
C Создать issue
Ctrl / Фокус на поиск issues/PR
U Фильтр по автору
L Фильтр или редактирование меток
M Фильтр или редактирование этапов
A Фильтр или редактирование исполнителя
O Открыть issue

#Задачи и pull requests

Сочетание клавиш Действие
Q Запросить ревью
M Установить этап
L Добавить метку
A Назначить исполнителя

#Изменения в пул-реквестах

Сочетание клавиш Действие
C Список коммитов в PR
T Список изменённых файлов в PR
J Выделить следующий в списке
K Выделить предыдущий в списке
Ctrl Shift Enter Добавить комментарий к diff
Alt (click) Свернуть/развернуть устаревшие комментарии ревью

#Доски проектов

Сочетание клавиш Действие
Enter/Space Начать перемещение колонки
Esc Отменить перемещение
Enter Завершить перемещение
Left/H Переместить колонку влево
Ctrl Left/H Переместить колонку в крайний левый
Right/L Переместить колонку вправо
Ctrl Right/L Переместить колонку в крайний правый
Enter/Space Начать перемещение карточки
Esc Отменить перемещение
Enter Завершить перемещение
Down/J Переместить карточку вниз
Ctrl Down/J Переместить карточку вниз колонки
Up/K Переместить карточку вверх
Ctrl Up/K Переместить карточку вверх колонки
Left/H Переместить карточку в левую колонку (вниз)
Shift Left/H Переместить карточку в левую колонку (вверх)
Ctrl Left/H Переместить в крайнюю левую колонку (вниз)
Ctrl Shift Left/H Переместить в крайнюю левую колонку (вверх)
Право Переместить карточку в правую колонку (вниз)
Shift Right/L Переместить карточку в правую колонку (вверх)
Ctrl Right/L Переместить в крайнюю правую колонку (вниз)
Ctrl Shift Right/L Переместить в крайнюю правую колонку (вверх)

#Уведомления

Сочетание клавиш Действие
