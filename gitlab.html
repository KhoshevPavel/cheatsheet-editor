GitLab cheatsheet

Наглядная шпаргалка по 58 горячим клавишам GitLab

#Горячие клавиши

#Глобальный

Сочетание клавиш Действие
? Показать/скрыть шпаргалку по горячим клавишам
Shift P Страница проектов
Shift G Страница групп
Shift A Страница активности
Shift L Страница этапов
Shift S Страница сниппетов
S Фокус на поиск issues или merge requests
Shift I Страница issues
Shift T Страница merge requests
P B Показать/скрыть панель производительности
Up Редактировать последний комментарий (в пустом поле под обсуждением)
Ctrl Shift P Предпросмотр Markdown при редактировании

#Проект

Сочетание клавиш Действие
G P Главная страница проекта
G V Лента активности проекта
G R Список релизов
G F Список файлов проекта
T Поиск файлов в проекте
G C Список коммитов
G N Граф репозитория
G D Графики репозитория
G I Список issues
I Создать новый issue
G B Доски issues
G M Список merge requests
G J Список CI/CD jobs
G L Метрики проекта
G E Окружения проекта
G K Интеграция Kubernetes
G S Список сниппетов
G W Вики проекта (если включена)

#Проблемы и мерж-реквесты

Сочетание клавиш Действие
E Редактировать описание
A Изменить исполнителя
M Изменить этап
L Изменить метку
R Написать комментарий (выделенный текст будет процитирован)
N Следующее нерешённое обсуждение (только MR)
P Предыдущее нерешённое обсуждение (только MR)
] Следующий файл (только MR)
[ Предыдущий файл (только MR)

#Файлы проекта

Сочетание клавиш Действие
Up Выделить выше
Down Выделить ниже
Enter Открыть выделенное
Esc Вернуться к списку файлов (при поиске)
Y Постоянная ссылка на файл (при просмотре)

#Веб-IDE

Сочетание клавиш Действие
Ctrl P Поиск и открытие файла для редактирования
Ctrl Enter Сделать коммит (при редактировании сообщения)

#Граф репозитория

Сочетание клавиш Действие
Лево Прокрутить влево
Право Прокрутить вправо
Up Прокрутить вверх
Down Прокрутить вниз
Shift Up Прокрутить вверх до конца
Shift Down Прокрутить вниз до конца

#Вики-страницы

Сочетание клавиш Действие
E Редактировать вики-страницу

#Фильтрованный поиск

Сочетание клавиш Действие
Ctrl Backspace Очистить весь фильтр поиска
Alt Backspace Удалять по одному токену

Сочетание клавиш Действие
R Написать комментарий (выделенный текст будет процитирован)

L Изменить метку
